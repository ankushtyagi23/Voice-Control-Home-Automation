# Voice-Control-Home-Automation
Voice Control Home Automation is a smart home system that allows users to control household appliances such as lights and fans using voice commands through a smartphone. The system is built using an Arduino microcontroller, Bluetooth communication, and relay modules to automate home devices wirelessly.

The project aims to improve convenience, accessibility, and energy efficiency by enabling users—especially elderly and physically challenged individuals—to operate home appliances without manual switches. Voice commands from an Android device are transmitted via Bluetooth to the Arduino board, which processes the commands and controls connected appliances accordingly.

According to the project documentation, the system uses a Bluetooth module to receive commands and a relay module to switch devices on or off, creating a simple and cost-effective home automation solution.

Key Features
Voice-controlled appliance management
Wireless control using Bluetooth
Arduino-based automation system
Easy-to-use Android interface
Low-cost and energy-efficient solution
Suitable for smart home and IoT applications

Technologies Used
Arduino Uno
Bluetooth Module (HC-05)
Relay Module
Android Smartphone
Arduino IDE (C/C++)
